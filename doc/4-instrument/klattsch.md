# klattsch instrument editor

the klattsch instrument editor only contains the following macros:

- **Volume**: volume sequence.
- **Arpeggio**: pitch sequence.
- **Pitch**: fine pitch.
- **Pan L**: left output level sequence.
- **Pan R**: right output level sequence.

phoneme selection, formant targets and voice quality are controlled with effect commands rather than instrument parameters. see the [klattsch](../7-systems/klattsch.md) system page for the full effect list and the phoneme table.
